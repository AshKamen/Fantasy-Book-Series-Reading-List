# Fantasy-Book-Series-Reading-List
A Python script that tracks book series (fantasy genre) that the user has read and generates a TBR list.
# How to use 
Run the script in your terminal, answer 'yes' or 'no' to the book series, and get a custom To-Be-Read list at the end.

# Why I created this project
Project created to practice using python. I practiced using lists and storing data, loops and string formatting.
